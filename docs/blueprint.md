# **App Name**: CRED

## Core Features:

- Wallet Connection: Wallet connection for deposit and withdrawal of funds.
- X Account Connection: X account connection for authorizing the app to read user activity.
- Tipping Rules Dashboard: Dashboard for managing funds and setting tipping rules (e.g., amount per like).
- Activity Monitoring: AI tool for monitoring X activity (likes) and triggering transactions to the escrow contract based on user-defined rules.
- Public Notification: Public tweet via quote tweet that is manually triggered in the beginning, crediting the Tipper and incentivizing the receiver.
- Claiming Page: Claiming page for new users to verify their X account, connect their wallet, and claim their tips.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to evoke trust and reliability, suitable for a financial application centered around social interactions.
- Background color: Light blue-gray (#F0F8FF) to provide a clean, unobtrusive backdrop that highlights the content. 
- Accent color: Electric purple (#7940FF) to draw attention to key interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, readable, and neutral look, suitable for both UI elements and potentially longer text blocks.
- Use simple, outline-style icons to maintain a clean and modern look.
- Maintain a clean, minimalist layout with a strong emphasis on readability and ease of use.
- Use subtle transitions and animations to enhance the user experience without being distracting.
